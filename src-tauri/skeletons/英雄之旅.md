# 英雄之旅（Hero's Journey/故事化叙事）

英雄之旅来自神话学研究，是经典的故事化叙事结构，情感穿透力强，能够建立深层品牌认同。

## 核心逻辑

平凡世界 → 遭遇挑战 → 遇见导师/获得工具 → 战胜困难 → 获得蜕变

## 适用场景

- 品牌故事宣讲
- 创始人分享
- 企业愿景发布会
- TED式演讲
- 变革动员

## 结构要点

1. **平凡世界**：描述初始状态，建立认同
2. **遭遇挑战**：危机、困境、转折点
3. **遇见导师/获得工具**：关键的启发或资源
4. **战胜困难**：核心的奋斗过程
5. **获得蜕变**：最终的成长和改变

## 优势

- 情感穿透力强
- 建立深层品牌认同
- 故事化表达，易于记忆和传播

## 劣势

- 对叙事能力要求极高
- 故事生硬易尴尬
- 不适合纯理性议题

## 示例

**平凡世界**：2015年，我们只是一个小县城的夫妻店，每天为几十个客户提供维修服务...

**遭遇挑战**：电商平台崛起，传统维修店生意一落千丈，房租都快交不起了...

**遇见导师/获得工具**：一个偶然的机会，我参加了一场关于"服务升级"的讲座，意识到：不是维修没市场，是服务需要升级...

**战胜困难**：我们开始做上门服务，建立服务标准，培训团队，一开始很难，客户不信任，员工不会做...

**获得蜕变**：三年后，我们成为本地最大的上门维修平台，团队从2人发展到50人，服务了上万个家庭...

现在，我们的使命是：让每一个家庭都能享受到专业的维修服务。
