PPT Builder

告别排版，回归思考

一款面向专业职场人的、支持私有化模型与品牌资产沉淀的 AI 结构化 PPT 引擎

了解更多

你是否经历过这样的时刻？

周五下午，老板突然说："下周一的战略汇报，你准备一下。"

然后你打开 PowerPoint，对着空白幻灯片发呆半小时——第一页该写什么？第二页该放什么数据？逻辑怎么串？

好不容易憋出内容，发现字体不对、颜色太艳、Logo 位置偏了。于是你开始"洗稿"——把 AI 生成的内容一个个复制粘贴到公司模板里，调格式、对齐、改颜色……

两个小时过去了，PPT 还没做完。

我们在解决什么问题？

第一件事

帮你梳理逻辑，不再对着空白发呆

你只需要告诉它："我要做一个季度战略汇报"，然后上传你的素材——Word 报告、Excel 数据、PDF 文献，甚至只是一段语音描述。

  • 提炼核心信息，生成结构化大纲
  • 确认逻辑框架后再动手，而不是黑盒生成
  • 每一页的标题、内容摘要、讲稿都帮你准备好
第二件事

设计生成美观的、符合自定义风格的 PPT

大纲确认后，PPT Builder 会为每一页生成精美的配图。

  • 支持上传企业模板作为参考，保持品牌一致性
  • 可自定义风格文件，定义配色、字体、布局要求
  • AI 会严格遵循你的风格规范，不会"发挥创意"
第三件事

导出即用，不用洗稿

大多数 AI PPT 工具有个致命问题：生成的 PPTX 不可编辑，或者格式一塌糊涂。

  • 文本框是真正的文本框，不是图片
  • 字体、颜色、Logo 位置严格遵循你上传的企业模板
  • 导出后直接提交，不需要任何格式清洗
第四件事

你的数据，只属于你

很多企业的商业计划书、财务数据，根本不敢放到公开的 AI 平台上。

  • 支持自带 API Key（BYOK），数据直接调用你选择的模型
  • 不经过我们的服务器存储
  • 你可以用 GPT-4o，也可以用文心一言、通义千问

功能特性

思路梳理（AI 对话）

  • 通过对话厘清演示目标和核心信息
  • 上传文本附件作为参考资料
  • AI 帮助梳理思路并生成需求文档

大纲生成

  • 智能大纲生成：基于需求文档自动生成结构化大纲
  • 汇报框架：支持 SCQA、STAR、PREP 等经典框架
  • 信息密度：简洁/标准/详细三种选择
  • 自由编辑：随时调整大纲内容

PPT 设计

  • 批量生成：一键为所有页面生成配图
  • 单页重生成：对不满意的单页可重新生成
  • 图片微调：支持对生成的图片进行微调
  • 模板参考：使用模板图片保持风格一致
  • 风格定制：自定义配色、风格等要求

导出分享

  • PDF 导出：一键导出高质量 PDF 文件
  • PPT 导出：一键导出可编辑的 PPT 文件
  • 原生格式：文本框可编辑，非图片

自定义配置

  • 大模型 API：自定义接入文生文和文生图 API
  • 模板管理：自定义多个模板按需选用
  • 风格管理：自定义多个风格按需选用
  • 框架管理：自定义汇报框架

支持的 AI 提供商

  • LLM：OpenAI、Azure、DashScope、DeepSeek、自定义
  • 图像生成：OpenAI、Azure、DashScope、自定义
  • 完全兼容 OpenAI API 格式

使用指南

1

配置 API

首次使用需要配置 LLM 和图像生成 API。点击左侧菜单「API」，填写提供商、API 端点、API Key 和模型名称，点击「测试连接」验证配置。

2

创建项目

点击左侧菜单「项目」，点击「新建项目」，输入项目主题，系统将自动创建项目文件夹。

3

思路梳理

进入「思路」页面，通过对话与 AI 交流，厘清演示目标。可上传文本附件作为参考资料，点击「生成需求」生成需求文档。

4

生成大纲

进入「大纲」页面，选择汇报框架、信息密度、期望页数，点击「生成大纲」。可直接编辑 Markdown 内容，点击「保存大纲」。

5

编辑页面与生成图片

进入「PPT设计」页面，选择模板和风格，点击「批量生成」或「生成当前」。生成完成后点击「导出 PDF」或「导出 PPT」生成最终文件。

PPT Builder 不是让你"玩 AI"，而是让你在遵守企业规范的前提下，把原本需要 3 天的汇报材料压缩到 3 小时完成。
