# PPT Builder - AI 结构化 PPT 引擎

> **告别排版，回归思考。**

**一款面向专业职场人的、支持私有化模型与品牌资产沉淀的 AI 结构化 PPT 引擎。**

---

## 你是否经历过这样的时刻？

周五下午，老板突然说："下周一的战略汇报，你准备一下。"

然后你打开 PowerPoint，对着空白幻灯片发呆半小时——**第一页该写什么？第二页该放什么数据？逻辑怎么串？**

好不容易憋出内容，发现字体不对、颜色太艳、Logo 位置偏了。于是你开始"洗稿"——把 AI 生成的内容一个个复制粘贴到公司模板里，调格式、对齐、改颜色……

**两个小时过去了，PPT 还没做完。**

这时候你一定想过：**"有没有一个工具，直接把我的素材变成合规的 PPT，不用我调格式？"**

**PPT Builder 就是这个工具。**

---

## 我们在解决什么问题？

我们不解决"从零开始做PPT"的问题，而是解决**"从素材到专业演示文稿的最后一公里"**：

### 第一件事：帮你梳理逻辑，不再对着空白发呆

你只需要告诉它：**"我要做一个季度战略汇报"**，然后上传你的素材——Word 报告、Excel 数据、PDF 文献，甚至只是一段语音描述。

它会帮你：
1. 提炼核心信息，生成结构化大纲
2. 确认逻辑框架后再动手，而不是黑盒生成
3. 每一页的标题、内容摘要、讲稿都帮你准备好

**你只需要在大纲阶段确认方向，剩下的交给它。**

### 第二件事：设计生成美观的、符合自定义风格的 PPT

大纲确认后，PPT Builder 会为每一页生成精美的配图。

- 支持上传企业模板作为参考，保持品牌一致性
- 可自定义风格文件，定义配色、字体、布局要求
- AI 会严格遵循你的风格规范，不会"发挥创意"

### 第三件事：导出即用，不用洗稿

大多数 AI PPT 工具有个致命问题：生成的 PPTX 不可编辑，或者格式一塌糊涂。

**PPT Builder 的导出是 100% 原生可编辑的。**

- 文本框是真正的文本框，不是图片
- 字体、颜色、Logo 位置严格遵循你上传的企业模板
- 导出后直接提交，**不需要任何格式清洗**

### 第四件事：你的数据，只属于你

很多企业的商业计划书、财务数据，根本不敢放到公开的 AI 平台上。

**PPT Builder 是本地优先的桌面应用。**

- 支持自带 API Key（BYOK），数据直接调用你选择的模型
- 不经过我们的服务器存储
- 你可以用 GPT-4o，也可以用文心一言、通义千问——**你的选择权在你手里**

## 功能特性

### 思路梳理（AI 对话）

- 通过对话厘清演示目标和核心信息
- 上传文本附件作为参考资料
- AI 帮助梳理思路并生成需求文档

### 大纲生成

- **智能大纲生成**：基于需求文档，AI 自动生成结构化大纲
- **汇报框架**：支持 SCQA、STAR、PREP 等经典汇报框架
- **信息密度**：简洁/标准/详细三种密度选择
- **自由编辑**：随时调整大纲内容

### PPT 设计

- **批量生成**：一键为所有页面生成配图
- **单页重生成**：对不满意的单页可重新生成
- **图片微调**：支持对生成的图片进行微调
- **模板参考**：使用模板图片作为参考，保持风格一致
- **风格定制**：自定义配色、风格等要求

### 导出分享

- **PDF 导出**：一键导出高质量 PDF 文件，方便分享和演示
- **PPT 导出**：一键导出可编辑的 PPT 文件，方便修订调整

### 自定义配置

- **大模型 API**：自定义接入文生文和文生图大模型 API
- **模板管理**：自定义多个模板按需选用，包含封面、内容页、封底
- **风格管理**：自定义多个风格按需选用，描述配色、风格等要求
- **框架管理**：自定义汇报框架，如 SCQA、STAR 等

## 安装

### Windows

下载 `PptBuilder_x.x.x_x64-setup.exe` 或 `PptBuilder_x.x.x_win64.zip` 安装包，双击安装。

### 从源码构建

#### 前置要求

1. **Node.js** 18+ 和 npm
2. **Rust** 工具链（通过 [rustup](https://rustup.rs/) 安装）
3. **LLVM/Clang**（用于编译 OCR 模块）

#### 安装 LLVM/Clang（Windows）

OCR 模块（`ocr-rs`）需要 `libclang` 来生成 Rust 绑定代码。

1. 下载 LLVM 安装程序：https://github.com/llvm/llvm-project/releases
   - 文件名类似：`LLVM-19.1.7-win64.exe`
2. 安装时勾选 **"Add LLVM to the system PATH"**
3. 或者手动设置环境变量：
   ```powershell
   $env:LIBCLANG_PATH = "C:\Program Files\LLVM\bin"
   ```

> **注意**：LLVM/Clang 仅在编译时需要，运行时不需要。用户安装编译好的程序时无需安装 LLVM。

#### 构建步骤

```bash
# 安装依赖
npm install

# 开发模式
npm run tauri dev

# 构建
npm run tauri build

# 一键打包（包含 exe + 数据文件夹）
npm run package
```

## 使用指南

### 1. 配置 API

首次使用需要配置 LLM 和图像生成 API：

1. 点击左侧菜单「API」
2. 填写 LLM 配置（用于生成大纲）
   - 提供商：OpenAI / Azure / DashScope / DeepSeek / 自定义
   - API 端点
   - API Key
   - 模型名称
3. 填写图像生成配置（用于生成图片）
4. 点击「测试连接」验证配置
5. 点击「保存配置」

### 2. 创建项目

1. 点击左侧菜单「项目」
2. 点击「新建项目」
3. 输入项目主题
4. 系统将自动创建项目文件夹

### 3. 思路梳理

1. 进入「思路」页面
2. 通过对话与 AI 交流，厘清演示目标
3. 可上传文本附件作为参考资料
4. 点击「生成需求」生成需求文档

### 4. 生成大纲

1. 进入「大纲」页面
2. 左侧显示需求内容，可自由编辑
3. 选择汇报框架、信息密度、期望页数
4. 点击「生成大纲」
5. 可直接编辑 Markdown 内容
6. 点击「保存大纲」

### 5. 设计页面与生成图片

1. 进入「PPT设计」页面
2. 左侧显示页面列表和 Markdown 编辑器
3. 右侧显示图片预览
4. 选择模板和风格
5. 点击「批量生成」或「生成当前」
6. 点击「导出 PDF」生成最终文件

### 模板定制

模板用于控制 PPT 的视觉风格和布局。每个模板是一个文件夹，包含以下文件：

| 文件 | 用途 | 必需 |
|------|------|------|
| `front-cover.png` | 封面参考图 | 是 |
| `content.png` | 内容页参考图 | 是 |
| `back-cover.png` | 封底参考图 | 是 |
| `template.pptx` | PPT 模板文件（用于导出可编辑 PPT） | 否 |

#### 参考图要求

参考图用于 AI 生成时参考风格：

- **尺寸**：推荐 1920×1080（16:9）
- **格式**：PNG 或 JPG
- **内容**：展示模板的整体布局、配色、字体风格

#### PPT 模板文件要求

`template.pptx` 用于「基于模板导出」功能，导出的 PPT 将继承模板的：

- **母版和主题**：配色方案、字体设置
- **幻灯片布局**：首页、内容页、尾页三种布局
- **幻灯片尺寸**：自动读取模板尺寸

**布局要求**：

模板 PPTX 应包含三种 slideLayout：

| 布局 | 类型名称 | 用途 |
|------|----------|------|
| slideLayout1 | title / cover / front | 封面页 |
| slideLayout2 | content / text | 内容页 |
| slideLayout3 | back / end | 封底页 |

如果模板没有命名布局，系统会自动按顺序分配：
- 第 1 页 → slideLayout1（封面）
- 第 2~N-1 页 → slideLayout2（内容）
- 第 N 页 → slideLayout3（封底）

**创建模板步骤**：

1. 在 PowerPoint 中创建新演示文稿
2. 设计母版（颜色、字体、背景）
3. 创建三种幻灯片布局
4. 保存为 `template.pptx`

**主题颜色**：

导出时会自动继承模板的主题颜色：
- `dk1` / `lt1`：深色/浅色
- `accent1-6`：强调色
- `hlink`：超链接颜色

文本框颜色会优先使用模板的 `dk1`（深色）。

**默认字体**：

导出时会自动继承模板的默认字体设置：
- 西文字体（latin）
- 东亚字体（ea）
